### 配置参数说明

| 修订时间 | 修订内容 |
| :--- | :--- |
| code    | 值为200，表示请求成功，会使用配置的链接 |
| address   | 值是跳转的网址链接，需要配置成自己的产品链接 |
| tColor  | 值是16进制，顶部状态栏颜色 |

---

`code` 的值为200，表示请求成功，会使用配置的链接

`mLink` 的值是跳转的网址链接，需要你配置成自己的产品链接

`tColor` 值是16进制，顶部状态栏颜色
